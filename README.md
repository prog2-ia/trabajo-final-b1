# Marketplace

Simulador de mercado. Permite gestionar un catálogo de productos y negociar precios con vendedores que poseen personalidades diferentes.

## Características
- **Jerarquía de Clases**: Categorías de Electrónica, Ropa, Hogar y Deportes.
- **Vendedores con personalidades**: Tipos Tacaño, Normal y Desesperado con frases diferentes.
- **Negociación Persistente**: Bucle de regateo que se mantiene activo hasta cerrar el trato.
- **Encapsulamiento**: Protección de precios mediante miembros privados y validación de datos.

## Estructura del Proyecto
- `datos.py`: Definición de productos y perfiles de vendedores.
- `logica.py`: Gestión del inventario y funciones de búsqueda.
- `interfaz.py`: Manejo de menús y mensajes de usuario.
- `mainpro.py`: Punto de entrada del programa.

## Cómo ejecutar el programa
Para iniciar el Marketplace, abre tu terminal en la carpeta del proyecto y ejecuta:

python mainpro.py
